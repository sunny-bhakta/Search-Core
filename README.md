## Recommended Learning Path

1. **TF-IDF**: Learn about term frequency, inverse document frequency, and cosine similarity (`01-tfidf/`).
2. **Fuzzy Search (Fuse.js)**: Explore fuzzy search and user experience improvements (`02-fuzzy-fuse/`).
3. **Embeddings**: Try semantic search using vector representations (`03-embeddings/`).
4. **Hybrid Search**: Combine lexical and semantic methods for better results (`04-hybrid/`).
5. **BM25**: Learn about the BM25 ranking algorithm (`05-bm25/`).
6. **N-gram & Prefix**: Use n-gram and prefix-based search for autocomplete and spell correction (`06-ngram-prefix/`).
7. **Boolean Filter**: Filter results using Boolean logic (`07-boolean-filter/`).
8. **Reranking**: Rerank results using additional signals or models (`08-reranking/`).
9. **Query Expansion**: Expand queries for better recall (`09-query-expansion/`).
10. **Boosters & Ranking**: Influence result order for relevance.
11. **Faceting**: Filter and refine results by attributes.
12. **Synonyms & Expansions**: Expand queries for better recall.
13. **Indexing**: Organize data for fast retrieval.
14. **Vector Search**: Use embeddings for semantic similarity.
15. **Elasticsearch**: Learn search engine fundamentals.
16. **PostgreSQL**: Explore search in relational databases.

Each folder has its own README and starter code. Tackle them in order, experiment, and compare results.
